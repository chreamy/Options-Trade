## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/chreamy/Options-Trade.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Options-Trade
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```
